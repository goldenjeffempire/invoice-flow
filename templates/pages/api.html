📋 Core Endpoints

POST /api/v1/invoices

Create a new invoice with client details, line items, and payment information.

GET /api/v1/clients

Retrieve a list of your clients or get details for a specific client.

POST /api/v1/payments

Record payments received against invoices with multiple payment methods.

GET /api/v1/reports

Get comprehensive reports on revenue, outstanding invoices, and cash flow.

POST /api/v1/webhooks

Subscribe to real-time events for invoices, payments, and other actions.

GET /api/v1/settings

Manage your account settings, currency, tax rates, and invoice templates.

⚡ Rate Limiting

Requests per minute: 60
Requests per hour: 1,000
Requests per day: 10,000

✓ Best Practices

  • Use HTTPS for all requests
  • Implement exponential backoff for retries
  • Cache responses when appropriate
  • Monitor webhook delivery
  • Rotate API keys regularly
