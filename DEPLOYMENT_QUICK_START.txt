# INVOICEFLOW - DEPLOYMENT QUICK START GUIDE

## For Immediate Production Deployment

### Step 1: Prepare Environment Variables
```bash
# Generate a secure SECRET_KEY
python -c "from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())"

# Generate a secure ENCRYPTION_SALT
python -c "import secrets; print(secrets.token_hex(32))"
```

### Step 2: Set Production Environment Variables
```bash
export PRODUCTION=true
export DEBUG=false
export SECRET_KEY="<your-generated-key>"
export DATABASE_URL="postgres://user:pass@host:5432/db"
export ENCRYPTION_SALT="<your-generated-salt>"
export ALLOWED_HOSTS="yourdomain.com,www.yourdomain.com"
export SENDGRID_API_KEY="<your-sendgrid-key>"
export SENTRY_DSN="<your-sentry-dsn>"
```

### Step 3: Prepare Database
```bash
python manage.py migrate
python manage.py collectstatic --noinput --clear
```

### Step 4: Start Production Server
```bash
gunicorn invoiceflow.wsgi:application \
  --bind 0.0.0.0:5000 \
  --workers 17 \
  --worker-class gthread \
  --threads 4 \
  --timeout 120 \
  --access-logfile - \
  --error-logfile - \
  --log-level info
```

### Step 5: Verify Deployment
```bash
# Health Check
curl http://localhost:5000/

# API Schema
curl http://localhost:5000/api/schema/

# Admin (if enabled)
curl http://localhost:5000/admin/
```

## Key Files to Review Before Deployment
- .env.example - Environment configuration template
- gunicorn.conf.py - Gunicorn production settings
- invoiceflow/settings.py - Django security settings
- FINAL_PRODUCTION_AUDIT.md - Complete audit report
- DEPLOYMENT_CHECKLIST.md - Full deployment checklist

## Production Server Specifications (Recommended)
- CPU: 2+ cores
- RAM: 2+ GB
- Storage: 10+ GB SSD
- Database: PostgreSQL 12+
- Python: 3.11+
- Load Balancer: nginx or Caddy
- Monitoring: Sentry + application logging
